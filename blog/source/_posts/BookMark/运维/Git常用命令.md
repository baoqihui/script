# Git常用命令
## git 提交
```
git clone https://gitee.com/idse666666/bbs.git
git remote -v
git remote rename origin hbq
git commit -m "less test" -a
git push origin hbq_dev:hbq_dev
```


## 清除git缓存
```
git rm -r --cached .
git add .
```
